# List open issues

!`gh issue list --state open`

---

## Display includes

- Issue number (#)
- Issue title
- State (Open/Closed)
- Labels (feat, fix, docs, etc.)
- Last update time

## Tip

Use issue numbers with `/git:start` to create branches from these issues
